# AI原生智能任务管理器 - 推广网页设计文档

> **设计日期**：2026年2月1日
> **产品版本**：v1.16.0
> **设计风格**：现代简洁，类似 Apple 产品页

---

## 一、项目概述

### 项目目标
为 AI原生智能任务管理器创建一个推广网页，支持中英双语，用于产品展示和用户获取。

### 产品信息
- **产品名称**：
  - 中文：AI原生智能任务管理器
  - 英文：AI-Native Smart Task Manager
- **核心价值**：让 AI 为你每月节省 20+ 小时，专注于真正重要的事
- **目标用户**：
  - 知识工作者（40%）：程序员、产品经理、设计师、研究人员
  - 自由职业者（30%）：独立开发者、设计师、撰稿人
  - 学生群体（20%）：大学生、研究生、博士生
  - 效率爱好者（10%）：生产力工具发烧友

---

## 二、技术方案

### 技术栈
- **HTML5**：语义化结构
- **Tailwind CSS**（CDN）：快速样式开发
- **Alpine.js**（CDN）：轻量级交互处理
- **Vanilla JavaScript**：语言切换、平滑滚动

### 部署方案
- **平台**：GitHub Pages
- **域名**：wuzhanmin.github.io
- **构建**：无需构建，直接推送 HTML 文件

### 语言切换方案
- **实现方式**：JavaScript 动态切换
- **默认语言**：中文
- **存储**：localStorage 保存用户选择
- **标记方式**：`data-lang="en"` 和 `data-lang="zh"`

## 六、交互设计

### 1. 平滑滚动
- 点击导航链接：平滑滚动到对应区域
- CSS：`scroll-behavior: smooth`
- JavaScript：确保偏移量正确（避开固定导航栏）

### 2. 滚动动画
- 元素进入视口时触发：淡入 + 上浮
- 使用 Intersection Observer API
- 动画时长：0.6s
- 缓动函数：ease-out

### 3. 按钮交互
- 悬停：颜色加深 10%
- 点击：轻微缩小（scale: 0.98）
- 焦点：显示蓝色轮廓

### 4. 卡片交互
- 悬停：上浮 4px + 阴影加深
- 过渡：0.3s ease-out

### 5. 语言切换
- 点击切换按钮：立即更新所有文本
- 保存到 localStorage：`lang = 'zh' | 'en'`
- 页面加载时：读取 localStorage，应用保存的语言

## 八、性能优化

### 图片优化
- 使用 WebP 格式（如果浏览器支持）
- 提供多种尺寸：响应式 `srcset`
- 懒加载：`loading="lazy"`

### 视频优化
- 视频封面图：避免加载前的空白
- 预加载策略：`preload="metadata"`
- 响应式：根据屏幕尺寸加载不同分辨率

### 字体优化
- 使用系统字体：无需加载外部字体文件
- `font-display: swap`（如果使用 Web Fonts）

### JavaScript 优化
- 按需加载：音频播放器代码延迟加载
- 代码压缩：使用 minified 版本
- 缓存策略：设置合适的 Cache-Control

## 九、可访问性（A11y）

### 语义化 HTML
- 使用 `<nav>`, `<header>`, `<main>`, `<section>`, `<footer>`
- 正确的标题层级：h1 > h2 > h3
- `<button>` vs `<a>`：正确使用

### 键盘导航
- 所有交互元素支持键盘访问
- 焦点可见：蓝色轮廓
- Tab 顺序：符合逻辑

### 屏幕阅读器
- `aria-label`：为图标按钮添加标签
- `alt` 文本：为所有图片提供描述
- `lang` 属性：正确标记中英文内容

### 颜色对比度
- 文字与背景对比度：至少 4.5:1（WCAG AA）
- 链接与背景对比度：至少 3:1
